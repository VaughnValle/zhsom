## Mod Feature List

This section lists the included sound improvements in the ZHSOM current build.

- Latest release: **1.0.0** (February 13, 2021)

### Weapon SFX

- Improved Kyanite SFX
- Improved Falcon SFX
- Improved G17 SFX
- Improved M4 SFX
  - M4 chamber
  - M4 mag eject and in
  - M4 slap
- Improved Mac-10 SFX
- Improved MP5 SFX
  - MP5 bolt pull
  - MP5 slap
- Improved SCAR 17 SFX
- Improved Shotgun SFX

### Grenade SFX
- Improved frag grenade SFX
- Improved flashband SFX

### Other Miscellaneous SFX
- Improved firemode change SFX
- Improved pistol SFX
  - Pistol chamber
  - Pistol mag eject and in
- Improved rifle SFX
  - Rifle empty Shot
  - Rifle mage eject and in
  - Rifle reload empty
